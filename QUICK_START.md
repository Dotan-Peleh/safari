# 🚀 מדריך התחלה מהירה

## שתי דרכים לשימוש:

### 1️⃣ אפליקציית React (מומלץ - אינטראקטיבי)

```bash
# התקנת תלויות
npm install

# הרצת האפליקציה
npm start
```

פותח את הדפדפן ב-`http://localhost:3000`

**תכונות:**
- ✅ מפה אינטראקטיבית
- ✅ בחירת מסלולים
- ✅ פרטים על כל יעד
- ✅ טיפים מפורטים
- ✅ השוואת מסלולים
- ✅ תמונות מקומיות (אין צורך באינטרנט!)

### 2️⃣ גרסה סטטית (HTML בלבד)

פשוט פתח את `index.html` בדפדפן.

**הערה:** התמונות נמצאות ב-`public/images/` - ודא שהנתיבים נכונים.

## 🖼️ תמונות

כל 16 התמונות נמצאות ב-`public/images/`:
- תמונות יעדים (בווינדי, קיבאלה, סרנגטי, נגורונגורו, טרנגירה)
- תמונות גלריה (6 תמונות)
- תמונות מסלולים (3 תמונות)
- תמונות רקע (2 תמונות)

## ✨ מה חדש?

✅ **קומפוננטת React מלאה** עם כל הפונקציונליות
✅ **תמונות מקומיות** - אין תלות באינטרנט
✅ **מפה אינטראקטיבית** - לחץ על יעדים לפרטים
✅ **3 מסלולים שונים** - מומלץ, חלופי, תקציבי
✅ **טיפים מפורטים** - לכל יעד ולכל נושא
✅ **השוואת מסלולים** - טבלה מפורטת

## 🎯 שימוש מהיר

1. **התקן:** `npm install`
2. **הרץ:** `npm start`
3. **בחר מסלול** מהכפתורים העליונים
4. **לחץ על יעד** במפה או ברשימה
5. **עיין בטאבים:** יעדים, טיפים, השוואה
