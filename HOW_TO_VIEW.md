## דרך 2: אפליקציית React (אינטראקטיבית) 🚀

### אם יש בעיית הרשאות ב-npm:

**נסה אחת מהאפשרויות:**

#### אופציה A: נקה את ה-cache של npm
```bash
npm cache clean --force
npm install
```

#### אופציה B: השתמש ב-yarn (אם מותקן)
```bash
yarn install
yarn start
```

#### אופציה C: התקן עם הרשאות
```bash
sudo npm install
npm start
```

### אחרי ההתקנה:
```bash
npm start
```

הדפדפן יפתח אוטומטית ב-`http://localhost:3000`

## פתרון בעיות תמונות

אם התמונות לא נטענות בגרסה הסטטית, ודא שהנתיבים נכונים:
- התמונות נמצאות ב: `public/images/`
- הקובץ HTML מפנה ל: `public/images/...`

אם זה לא עובד, שנה את הנתיבים ב-HTML מ-`public/images/` ל-`images/` והזז את התיקייה.
