## 🚀 התקנה והרצה מקומית

### דרישות מוקדמות
- Node.js (גרסה 14 ומעלה)
- yarn או npm

### התקנה
```bash
# Clone the repository
git clone https://github.com/your-username/safari-trip-planner.git
cd safari-trip-planner

# Install dependencies
yarn install
# או
npm install
```

### הרצה
```bash
yarn start
# או
npm start
```

האפליקציה תיפתח אוטומטית ב-`http://localhost:3000`

### בנייה לייצור
```bash
yarn build
# או
npm run build
```

## 🚀 פריסה ב-Netlify

הפרויקט מוכן לפריסה ב-Netlify:

1. **דחוף ל-GitHub** (ראה הוראות למטה)
2. **התחבר ל-Netlify** → [netlify.com](https://netlify.com)
3. **New site from Git** → בחר את ה-repo
4. **Build settings:**
   - Build command: `yarn build`
   - Publish directory: `build`
5. **Deploy!**

הקובץ `netlify.toml` כבר מוגדר עם כל ההגדרות הנדרשות.
